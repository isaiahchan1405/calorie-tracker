This is the repository for MealLogger, a web-based application that eases the process of tracking meals during the day. 

The interface is very easy to use and has all the necessary food options for our users to choose from. 

Our users can select the amount of a selected food they consumed, and our app will calculate and return the macros. 

MealLogger also has the option to calculate the necessary daily intake based on the measurements and needs of the user.
